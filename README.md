# Weather Module

A containerized weather dashboard for Greek cities. The project combines a Python FastAPI WebSocket service, a Spring Boot Kafka bridge, a Kafka broker, and a Vite React dashboard.

## Highlights

- Live current weather readings from Open-Meteo
- Short-term local prediction stream generated by a small RandomForest model
- Spring Boot bridge that publishes weather events into Kafka topics
- React dashboard with responsive weather cards
- One-command Docker Compose startup

## Architecture

```text
Browser UI
   |
   | ws://localhost:8000/ws
   v
Python FastAPI weather service
   |
   | current weather: Open-Meteo
   | predictions: local ML model
   v
Spring Boot WebSocket subscriber
   |
   v
Kafka topics
   |-- weather.current
   |-- weather.predictions
   |-- weather.metadata
```

## Quick Start

Requirements:

- Docker Desktop

Run the full stack:

```powershell
docker compose up --build
```

Open the dashboard:

```text
http://localhost:5173
```

## Services

| Service | URL | Purpose |
| --- | --- | --- |
| React dashboard | `http://localhost:5173` | Weather UI |
| Python FastAPI | `http://localhost:8000` | Current weather and WebSocket stream |
| Python health | `http://localhost:8000/health` | Service health check |
| Spring Boot | `http://localhost:8080` | Kafka bridge |
| Kafka | `localhost:9092` | Event broker |

## Docker Commands

Start already-built containers:

```powershell
docker compose up
```

Run in the background:

```powershell
docker compose up -d
```

Stop without deleting containers:

```powershell
docker compose stop
```

Start stopped containers:

```powershell
docker compose start
```

Stop and remove containers:

```powershell
docker compose down
```

Stop and remove the Kafka volume:

```powershell
docker compose down -v
```

View logs:

```powershell
docker compose logs -f
```

## Manual Development

Start Kafka only:

```powershell
docker compose up -d kafka
```

Run the Python service:

```powershell
cd pythonML
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r req.txt
python initialize_model_py.py
python websocketService.py
```

Run the Spring Boot bridge:

```powershell
cd webSocketServer
.\gradlew.bat bootRun
```

Run the React dashboard:

```powershell
cd webSocketServer\src\main\client
npm install
npm run dev
```

## Data Notes

- Current weather comes from Open-Meteo.
- The Python service caches current weather for 60 seconds to keep page reloads fast.
- If Open-Meteo is unavailable, the service falls back to generated weather values.
- The prediction model is trained from synthetic data using `initialize_model_py.py`.
- The generated `weather_model.joblib` file is not required in Git because it can be rebuilt.
